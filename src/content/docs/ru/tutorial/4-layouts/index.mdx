Теперь, когда вы можете создавать компоненты, давайте создадим некоторые пользовательские макеты!

## Где мы сейчас?

Вы можете найти код на этом этапе учебника на [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-4/start) или [StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-4/start?file=src/pages/index.astro).

Сравните ваш код с этим примером, или, если вы только что присоединились к нам, создайте форк на StackBlitz, чтобы начать кодировать прямо в вашем браузере отсюда.

## Куда мы идем?

В этом модуле вы создадите макеты для общих элементов и стилей на всех своих страницах и сообщениях блога.

Для этого вы будете:

- Создавать многоразовые компоненты макета
- Передавать контент на свои макеты с помощью `<slot />`
- Передавать данные из frontmatter Markdown на свои макеты
- Вкладывать несколько макетов
