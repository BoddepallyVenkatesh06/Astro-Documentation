Здесь вы подключите ваш репозиторий на GitHub к Netlify. Netlify будет использовать этот проект для сборки и развертывания вашего сайта в реальном времени на веб-сервере каждый раз, когда вы вносите изменения в свой код.
:::tip[Мы будем использовать…]
В этом учебном пособии мы будем использовать **Netlify**, но вы можете использовать любой другой выбранный вами хостинг-сервис для развертывания вашего сайта в интернете.
:::

## Создание нового сайта на Netlify

1. Создайте бесплатную учетную запись на Netlify, если у вас еще ее нет.

    Запишите свое имя пользователя. Вы можете просмотреть свою панель инструментов и любые созданные сайты по адресу `https://app.netlify.com/teams/username`

2. Нажмите <kbd>Добавить новый сайт</kbd> > <kbd>Импортировать существующий проект</kbd>.

    Вам будет предложено подключиться к Git-провайдеру. Выберите GitHub и следуйте инструкциям на экране, чтобы аутентифицировать свою учетную запись GitHub. Затем выберите репозиторий вашего проекта Astro на GitHub из предоставленного списка.

3. На последнем шаге Netlify покажет вам настройки вашего сайта. Параметры по умолчанию должны быть правильными для вашего проекта Astro, так что вы можете прокрутить вниз и нажать <kbd>Развернуть сайт</kbd>.

Поздравляем, у вас есть сайт Astro!

## Измените название вашего проекта

На странице обзора вашего сайта в Netlify вы увидите случайно сгенерированное название проекта и URL вашего сайта в формате `https://project-name-123456.netlify.app`. Вы можете изменить название вашего проекта на более запоминающееся, и это автоматически обновит ваш URL.

## Посетите ваш новый сайт

Щелкните URL-адрес в настройках вашего сайта или введите его в окно браузера, чтобы просмотреть ваш новый сайт.
