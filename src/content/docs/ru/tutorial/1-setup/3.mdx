## Отредактируйте главную страницу

1. В своем редакторе перейдите в файл `src/pages/index.astro` через панель навигации по файлам и щелкните на нем, чтобы открыть содержимое файла в редактируемой вкладке.

    Содержимое вашего файла `index.astro` должно выглядеть примерно так:

    ```astro title="src/pages/index.astro"
    ---
    ---

    <html lang="ru">
      <head>
        <meta charset="utf-8" />
        <link rel="icon" type ="image/svg+xml" href="/favicon.ico" />
        <meta name="viewport" content="width=device-width" />
        <meta name="generator" content = {Astro.generator} >
        <title>Astro</title>
      </head>
      <body>
        <h1>Astro</h1>
      </body>
    </html>
    ```

2. Отредактируйте содержимое вашей страницы в теге `<body>`.

    Наберите текст в редакторе для изменения заголовка на вашей странице и сохраните изменение.

    ```astro title="src/pages/index.astro" del={2} ins={3}
    <body>
      <h1>Astro</h1>
      <h1>Мой Astro сайт</h1>
    </body>
    ```

3. Проверьте предварительный просмотр в браузере, и вы должны увидеть, что содержимое вашей страницы обновилось с новым текстом.

Поздравляем! Теперь вы являетесь разработчиком Astro!

Остальная часть этого блока обучения настроит вас на версионный контроль и опубликует ваш сайт, который вы сможете показать другим.
