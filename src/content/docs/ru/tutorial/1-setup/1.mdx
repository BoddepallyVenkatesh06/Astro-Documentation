## Получите нужные для разработки инструменты

### Терминал

Вы будете использовать **командную строку (терминал)**, чтобы создать свой проект на Astro и запускать ключевые команды для сборки, разработки и проверки вашего сайта.

Вы можете получить доступ к командной строке через локальную программу для терминала вашей операционной системы. Обычные приложения включают **Terminal** (MacOS / Linux), **Command Prompt** (Windows) и **Termux** (Android). Одно из них, вероятно, уже есть на вашем компьютере.

### Node.js

Для работы Astro на вашей системе вам также потребуется установить [**Node.js**](https://nodejs.org/ru/), версия v18.14.1 или выше.

Чтобы проверить, установлена ли на вашем компьютере совместимая версия, выполните следующую команду в терминале:

```sh
node -v

// Пример вывода
v18.14.1
```
Если команда возвращает номер версии выше, чем `v18.14.1`, вы готовы к работе!

Если команда возвращает сообщение об ошибке типа `Command 'node' not found` или номер версии ниже, чем `v18.14.1`, то вам нужно [установить совместимую версию Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Редактор кода

Кроме того, вам нужно скачать и установить **редактор кода** для написания вашего кода.

:::tip[мы используем…]
Это учебное пособие использует **VS Code**, но вы можете использовать любой редактор для вашей операционной системы.
:::

1. [Скачайте и установите VS Code](https://code.visualstudio.com/#alt-downloads) или другой редактор кода на ваш выбор.
