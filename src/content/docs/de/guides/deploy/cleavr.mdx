Du kannst dein Astro-Projekt mit [Cleavr](https://cleavr.io/), einem Tool zur Verwaltung von Servern und Apps, auf deinem eigenen Virtual Private Server (VPS) veröffentlichen.

:::tip
Schau dir [den Astro-Leitfaden in den Cleavr-Dokumenten](https://docs.cleavr.io/guides/astro) an!
:::

## Voraussetzungen

Um loszulegen brauchst du Folgendes:

- Ein Cleavr-Konto
- Ein Server bei deinem VPS-Anbieter mit Cleavr

## Füge deine Seite hinzu

1. Navigiere in Cleavr zu dem Server, zu dem du dein Astro-Projekt hinzufügen möchtest.

2. Wähle **Seite hinzufügen** und gib die Details für deinen Antrag ein, z. B. den Domainnamen.

3. Bei **App-Typ** wählst du "NodeJS Static" oder "NodeJS SSR", je nachdem, wie du deine Astro-App einrichtest.
   
4. Für statische Anwendungen stellst du **Artifact Folder** auf `dist`.
   
5. Für SSR-Anwendungen:
   - Setze den **Einstiegspunkt** auf `Einstieg.mjs`.
   - Setze **Artifact Folder** auf `dist/server`.
   
6. Wähle **Hinzufügen**, um die Seite zu deinem Server hinzuzufügen.


## Einrichten und veröffentlichen

1. Sobald deine neue Website hinzugefügt wurde, klicke auf **Einrichten und bereitstellen**.
   
2. Wähle das **VC-Profil**, **Repo** und **Branchen** für dein Astro-Projekt.
   
3. Nimm alle zusätzlichen Konfigurationen vor, die für dein Projekt notwendig sind.
   
4. Klicke auf die Registerkarte **Deployment** und dann auf **Deploy**.

Herzlichen Glückwunsch, du hast soeben deine Astro-App veröffentlicht!
