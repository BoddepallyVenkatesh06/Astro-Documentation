[Spinal](https://spinalcms.com/cms-for-astro/) 是一款商业化的、专注于 SaaS、基于 Git 的 CMS。

## 快速入门

1. [创建一个 Spinal 账户](https://spinalcms.com/signup/)。
2. 连接你的 GitHub 和 Spinal 账户。
3. 选择你的 Astro 项目存储库。

在完成上述步骤后，在你所选文件夹中所有的 Markdown 内容都会被导入到你的 Spinal 账户，你可以 Spinal 编辑器中直接编辑。
