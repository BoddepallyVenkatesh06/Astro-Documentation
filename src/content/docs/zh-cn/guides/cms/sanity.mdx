[Sanity](http://sanity.io) 是一款无头 CMS，其专注于管理[结构化的内容](https://www.sanity.io/structured-content-platform).

## 官方资源

- [使用 Sanity 作为 CMS 来构建你的 Astro 博客（英文）](https://www.sanity.io/guides/sanity-astro-blog)

## 社区资源

- 使用 [`astro-sanity` 集成](https://github.com/littlesticks/astro-sanity)来向你的 Astro 项目添加 Sanity。

- 参考 [Astro Sanity Minimal Starter](https://github.com/littlesticks/astro-sanity-minimal-starter)。
