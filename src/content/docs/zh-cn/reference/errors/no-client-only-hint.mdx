> **NoClientOnlyHint**: 当使用 `client:only` 激活组件时，Astro 需要一个提示来使用正确的渲染器，否则无法渲染组件 `COMPONENT_NAME`。

## 哪里出了问题？

`client:only` 组件不会在服务器上运行，因此 Astro 不知道（也猜不到）要使用哪个渲染器，并且需要一个提示。如下所示：

```astro
 <SomeReactComponent client:only="react" />
```
