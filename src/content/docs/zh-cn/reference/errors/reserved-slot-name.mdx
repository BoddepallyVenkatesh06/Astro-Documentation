> **ReservedSlotName**: 无法创建名为 `SLOT_NAME` 的插槽。 `SLOT_NAME` 是保留的插槽名称。请更改此插槽的名称。

## 哪里出了问题？

某些词语由于已经被内部占用，因此不能作为插槽名称使用。

**请参阅：**

- [命名插槽](/zh-cn/core-concepts/astro-components/#命名插槽)
