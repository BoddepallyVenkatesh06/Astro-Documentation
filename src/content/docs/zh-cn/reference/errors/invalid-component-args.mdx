> **错误信息示例：**<br/>
InvalidComponentArgs: 向 `<MyAstroComponent>` 组件传递了无效的参数。

## 哪里出了问题？

Astro 组件不能通过函数调用的方式手动渲染，如 `Component()` 或 `{items.map(Component)}`。请使用组件语法 `<Component />` 或 `{items.map(item => <Component {...item} />)}`。
