> **错误信息示例:**<br/>
无法解析位于 **blog** → **post.md** 文件中的 frontmatter。<br/>
需要提供 "title" 字段。<br/>
"date" 必须是一个有效的日期值。

## 哪里发生了错误？

位于 `src/content/` 目录下的 Markdown 文档的 frontmatter 与集合模式不匹配。

请确保所有必需的字段都存在，并且所有字段的类型正确。

你可以在 `src/content/config.*` 文件中检查集合模式。

阅读更多有关于 [内容集合文档](/zh-cn/guides/content-collections/) 的内容。
