## 哪里发生了错误？

Astro 加载配置文件时遇到了一个未知错误。这通常是由于你的配置中存在语法错误，该消息应该提供更多信息。

如果你能够稳定的复现这个错误，[提交这个问题](https://astro.build/issues/)我们将不胜感激。
