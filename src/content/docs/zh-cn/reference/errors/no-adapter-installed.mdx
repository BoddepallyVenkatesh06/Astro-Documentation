> **NoAdapterInstalled**: 在没有适配器的情况下，无法使用 `output: 'server'` 或 `output: hybrid`。请为你的部署目标安装并配置适当的服务器适配器。

## 哪里出了问题？

要使用服务器端渲染，需要安装适配器，这样 Astro 才能知道如何为你的目标部署平台生成正确的输出。

**请参阅：**

- [服务端渲染](/zh-cn/guides/server-side-rendering/)
