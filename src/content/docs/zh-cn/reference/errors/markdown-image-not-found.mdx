> 未能在 `FULL_IMAGE_PATH` 找到所请求的图片 `IMAGE_PATH`。

## 哪里发生了错误？

Astro 无法找到在 Markdown 内容中包含的图像。通常是由路径中的拼写错误引起的。

Markdown 中的图像是相对于当前文件的。要引用与 `.md` 文件位于同一文件夹中的图像，路径应该以 `./` 开头。

**请参见：**

- [图像](/zh-cn/guides/images/)
