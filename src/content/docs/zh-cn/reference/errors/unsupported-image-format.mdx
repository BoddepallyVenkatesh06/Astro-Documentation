> **UnsupportedImageFormat**: 图片路径 `IMAGE_PATH` 所提供的图像格式 `FORMAT` 不受支持。目前我们的图片服务只支持 SUPPORTED_FORMATS.JOIN(', ')。

## 哪里出了问题？

内置的图像服务当前并不支持优化所有图像格式。

对于不支持的格式，如 GIF，你可以直接使用 `img` 标签：

```astro
---
import rocket from '../assets/images/rocket.gif';
---

<img src={rocket.src} width={rocket.width} height={rocket.height} alt="A rocketship in space." />
```
