> 未知的编译器错误。

## 哪里出了问题？
Astro 在编译文件时遇到了未知错误。在大多数情况下，这不是你的错，而是编译器的问题。

如果现有的 issue 中没有提到这个问题，请[创建一个 issue](https://astro.build/issues/compiler)。

**请参阅：**
-  [withastro/compiler的 issue 清单](https://astro.build/issues/compiler)
