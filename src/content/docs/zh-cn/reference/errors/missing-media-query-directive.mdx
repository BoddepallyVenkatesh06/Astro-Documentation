> **MissingMediaQueryDirective**: 没有为 `client:media` 指令提供媒体查询条件。必须提供类似于 `client:media="(max-width: 600px)"` 的媒体查询

## 哪里出了问题？
使用 `client:media` 指令时，需要提供 [媒体查询](https://developer.mozilla.org/zh-CN/docs/Web/CSS/Media_Queries/Using_media_queries) 参数。

```astro
<Counter client:media="(max-width: 640px)" />
```

**请参阅：**
-  [`client:media`](/zh-cn/reference/directives-reference/#clientmedia)
