> **ImageNotFound**: 未能找到所请求图片的 `IMAGE_PATH`。请确认该图像是否存在？

## 哪里出了问题？

Astro 找不到你所导入的图片。通常，这可能是由路径拼写错误引起的。

在 Markdown 中的图像是相对于当前文件的。要引用与 `.md` 文件位于同一文件夹中的图像，路径应该以 `./` 开头。

**请参阅：**
-  [Images](/zh-cn/guides/images/)
