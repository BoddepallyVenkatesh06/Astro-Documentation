> **MdxIntegrationMissingError**: 不能够渲染 FILE。请确保已经安装了 `@astrojs/mdx` 集成。

## 哪里发生了错误？

不能够找到官方的 `@astrojs/mdx` 集成。当使用 MDX 文件而没有安装 MDX 集成时，会引发此错误。

**请参阅：**

- [MDX 的安装与使用](/zh-cn/guides/integrations-guide/mdx/)
