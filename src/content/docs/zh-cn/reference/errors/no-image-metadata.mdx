> 不能处理 `IMAGE_PATH` 图像的元数据。

## 哪里发生了问题？

Astro 无法处理你导入的图像的元数据。这通常是因为图像损坏或格式错误所导致的，尝试从图像编辑器中重新导出图像可能会解决该问题。
