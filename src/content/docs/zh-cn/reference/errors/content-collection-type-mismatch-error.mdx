> `COLLECTION_NAME` 包含 `ACTUAL_TYPE` 类型的条目，但却配置为 `EXPECTED_TYPE` 集合。

## 哪里发生了错误？

内容集合必须包含所配置的类型。默认类型为 `type: 'content'`。试着为数据集合添加 `type: 'data'` 的类型。

**请参阅：**

- [定义内容集合](/zh-cn/guides/content-collections/#定义集合)
