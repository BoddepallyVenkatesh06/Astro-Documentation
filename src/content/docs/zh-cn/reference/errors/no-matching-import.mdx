> **NoMatchingImport**: 无法渲染 `COMPONENT_NAME` 组件。找不到与 `COMPONENT_NAME` 组件匹配的导入。

## 哪里出了问题？

找不到某个组件的导入语句。如果你已经导入了这个组件，请确保你在导入语句和使用组件的地方使用的标识符是一致的。
