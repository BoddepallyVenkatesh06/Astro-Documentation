Você pode fazer deploy do seu projeto Astro para a [Edgio](https://www.edg.io/), uma plataforma na edge e de CDN para fazer deploy, proteger e acelerar websites e APIs.

:::tip
Veja o [guia para Astro na documentação da Edgio](https://docs.edg.io/guides/astro)!
:::

## Como fazer o deploy

1. Instale [a CLI da Edgio](https://docs.edg.io/guides/cli) globalmente pelo Terminal, se você já não tiver.

    ```bash
    npm install -g @edgio/cli
    ```

2. Adicione Edgio ao seu site Astro

    ```bash
    edgio init
    ```

3. (Opcional) Habilite Renderização no Lado do Servidor

    Após você ter configurado [@astrojs/node com o Astro](/pt-br/guides/integrations-guide/node/), especifique o caminho do arquivo do servidor em `edgio.config.js` como abaixo:

    ```js ins={2,5-9}
    // edgio.config.js
    import { join } from 'path'

    module.exports = {
      astro: {
        // O caminho do servidor standalone que executa o SSR do Astro.
        // As dependências para este arquivo passam por bundle automaticamente.
        appPath: join(process.cwd(), 'dist', 'server', 'entry.mjs'),
      },
    };
    ```

4. Faça deploy para a Edgio

    ```bash
    edgio deploy
    ```
