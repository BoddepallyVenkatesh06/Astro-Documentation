Agora que você pode construir com componentes, é hora de criar alguns layouts customizados!

## Olhando à frente

Nesta unidade, você irá construir layouts para compartilhar elementos e estilos comuns entre suas páginas e postagens do blog.

Para fazer isso, você irá:

- Criar componentes reutilizáveis de layout
- Passar conteúdo para seus layouts com `<slot />`
- Passar dados do frontmatter do Markdown para seus layouts
- Aninhar múltiplos layouts
