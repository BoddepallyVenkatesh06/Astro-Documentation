Agora que você tem um site funcional na web, é hora de adicionar páginas e postagens!

## Olhando à frente

Nesta unidade, você irá:
- Criar suas primeiras páginas Astro com a sintaxe `.astro`
- Adicionar postagens ao blog com arquivos Markdown (`.md`)
- Estilizar uma página individual com `<style>` 
- Aplicar estilos globais entre páginas

 Durante o caminho, você aprenderá como as **duas seções de um arquivo `.astro`**  funcionam juntas para criar uma página, e como utilizar variáveis e renderização condicional em suas páginas.
