A referência a seguir é uma lista completa dos erros que você pode encontrar enquanto utiliza Astro. Para mais ajuda, incluindo pegadinhas comuns, por favor também veja nosso [Guia de Solução de Problemas](/pt-br/guides/troubleshooting/).

## Erros do Astro

- [**UnknownCompilerError**](/pt-br/reference/errors/unknown-compiler-error/)<br/>Unknown compiler error.
- [**StaticRedirectNotAvailable**](/pt-br/reference/errors/static-redirect-not-available/)<br/>`Astro.redirect` is not available in static mode.
- [**ClientAddressNotAvailable**](/pt-br/reference/errors/client-address-not-available/)<br/>`Astro.clientAddress` is not available in current adapter.
- [**StaticClientAddressNotAvailable**](/pt-br/reference/errors/static-client-address-not-available/)<br/>`Astro.clientAddress` is not available in static mode.
- [**NoMatchingStaticPathFound**](/pt-br/reference/errors/no-matching-static-path-found/)<br/>No static path found for requested path.
- [**OnlyResponseCanBeReturned**](/pt-br/reference/errors/only-response-can-be-returned/)<br/>Invalid type returned by Astro page.
- [**MissingMediaQueryDirective**](/pt-br/reference/errors/missing-media-query-directive/)<br/>Missing value for `client:media` directive.
- [**NoMatchingRenderer**](/pt-br/reference/errors/no-matching-renderer/)<br/>No matching renderer found.
- [**NoClientEntrypoint**](/pt-br/reference/errors/no-client-entrypoint/)<br/>No client entrypoint specified in renderer.
- [**NoClientOnlyHint**](/pt-br/reference/errors/no-client-only-hint/)<br/>Missing hint on `client:only` directive.
- [**InvalidGetStaticPathParam**](/pt-br/reference/errors/invalid-get-static-path-param/)<br/>Invalid value returned by a `getStaticPaths` path.
- [**InvalidGetStaticPathsEntry**](/pt-br/reference/errors/invalid-get-static-paths-entry/)<br/>Invalid entry inside getStaticPath's return value
- [**InvalidGetStaticPathsReturn**](/pt-br/reference/errors/invalid-get-static-paths-return/)<br/>Invalid value returned by getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/pt-br/reference/errors/get-static-paths-removed-rsshelper/)<br/>getStaticPaths RSS helper is not available anymore.
- [**GetStaticPathsExpectedParams**](/pt-br/reference/errors/get-static-paths-expected-params/)<br/>Missing params property on `getStaticPaths` route.
- [**GetStaticPathsInvalidRouteParam**](/pt-br/reference/errors/get-static-paths-invalid-route-param/)<br/>Invalid value for `getStaticPaths` route parameter.
- [**GetStaticPathsRequired**](/pt-br/reference/errors/get-static-paths-required/)<br/>`getStaticPaths()` function required for dynamic routes.
- [**ReservedSlotName**](/pt-br/reference/errors/reserved-slot-name/)<br/>Invalid slot name.
- [**NoAdapterInstalled**](/pt-br/reference/errors/no-adapter-installed/)<br/>Cannot use Server-side Rendering without an adapter.
- [**NoMatchingImport**](/pt-br/reference/errors/no-matching-import/)<br/>No import found for component.
- [**InvalidPrerenderExport**](/pt-br/reference/errors/invalid-prerender-export/)<br/>Invalid prerender export.
- [**InvalidComponentArgs**](/pt-br/reference/errors/invalid-component-args/)<br/>Invalid component arguments.
- [**PageNumberParamNotFound**](/pt-br/reference/errors/page-number-param-not-found/)<br/>Page number param not found.
- [**ImageMissingAlt**](/pt-br/reference/errors/image-missing-alt/)<br/>Missing alt property.
- [**InvalidImageService**](/pt-br/reference/errors/invalid-image-service/)<br/>Error while loading image service.
- [**MissingImageDimension**](/pt-br/reference/errors/missing-image-dimension/)<br/>Missing image dimensions
- [**UnsupportedImageFormat**](/pt-br/reference/errors/unsupported-image-format/)<br/>Unsupported image format
- [**UnsupportedImageConversion**](/pt-br/reference/errors/unsupported-image-conversion/)<br/>Unsupported image conversion
- [**PrerenderDynamicEndpointPathCollide**](/pt-br/reference/errors/prerender-dynamic-endpoint-path-collide/)<br/>Prerendered dynamic endpoint has path collision.
- [**ExpectedImage**](/pt-br/reference/errors/expected-image/)<br/>Expected src to be an image.
- [**ExpectedImageOptions**](/pt-br/reference/errors/expected-image-options/)<br/>Expected image options.
- [**IncompatibleDescriptorOptions**](/pt-br/reference/errors/incompatible-descriptor-options/)<br/>Cannot set both `densities` and `widths`
- [**ImageNotFound**](/pt-br/reference/errors/image-not-found/)<br/>Image not found.
- [**NoImageMetadata**](/pt-br/reference/errors/no-image-metadata/)<br/>Could not process image metadata.
- [**MarkdownImageNotFound**](/pt-br/reference/errors/markdown-image-not-found/)<br/>Image not found.
- [**ResponseSentError**](/pt-br/reference/errors/response-sent-error/)<br/>Unable to set response.
- [**MiddlewareNoDataOrNextCalled**](/pt-br/reference/errors/middleware-no-data-or-next-called/)<br/>The middleware didn't return a response or call `next`.
- [**MiddlewareNotAResponse**](/pt-br/reference/errors/middleware-not-aresponse/)<br/>The middleware returned something that is not a `Response` object.
- [**LocalImageUsedWrongly**](/pt-br/reference/errors/local-image-used-wrongly/)<br/>ESM imported images must be passed as-is.
- [**LocalsNotAnObject**](/pt-br/reference/errors/locals-not-an-object/)<br/>Value assigned to `locals` is not accepted.
- [**AstroGlobUsedOutside**](/pt-br/reference/errors/astro-glob-used-outside/)<br/>Astro.glob() used outside of an Astro file.
- [**AstroGlobNoMatch**](/pt-br/reference/errors/astro-glob-no-match/)<br/>Astro.glob() did not match any files.
- [**RedirectWithNoLocation**](/pt-br/reference/errors/redirect-with-no-location/)<br/>A redirect must be given a location with the `Location` header.
- [**InvalidDynamicRoute**](/pt-br/reference/errors/invalid-dynamic-route/)<br/>Invalid dynamic route.
- [**MissingSharp**](/pt-br/reference/errors/missing-sharp/)<br/>Could not find Sharp.
- [**UnknownViteError**](/pt-br/reference/errors/unknown-vite-error/)<br/>Unknown Vite Error.
- [**FailedToLoadModuleSSR**](/pt-br/reference/errors/failed-to-load-module-ssr/)<br/>Could not import file.
- [**InvalidGlob**](/pt-br/reference/errors/invalid-glob/)<br/>Invalid glob pattern.
- [**FailedToFindPageMapSSR**](/pt-br/reference/errors/failed-to-find-page-map-ssr/)<br/>Astro couldn't find the correct page to render
