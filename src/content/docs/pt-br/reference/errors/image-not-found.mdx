> **ImageNotFound**: Não foi possível encontrar a imagem requisitada `CAMINHO_IMAGEM`. Ela existe?

## O que deu errado?
O Astro não conseguiu encontrar uma imagem que você importou. Muitas vezes, isso ocorre simplesmente devido a um erro de digitação do caminho.

As imagens em Markdown são relativas ao arquivo atual. Para se referir a uma imagem que está localizada na mesma pasta que o arquivo `.md`, o caminho deve começar com `./`
