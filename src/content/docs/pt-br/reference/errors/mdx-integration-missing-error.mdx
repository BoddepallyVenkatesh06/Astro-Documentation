> **MdxIntegrationMissingError**: Incapaz de renderizar `NOME_ARQUIVO`. Certifique-se de que a integração `@astrojs/mdx` está instalada.

## O que deu errado?

Foi incapaz de encontrar a integração `@astrojs/mdx`. Esse erro acontece ao utilizar arquivos MDX sem a integração para MDX instalada.
