## O que deu errado?

Vite encontrou um erro desconhecido enquanto renderizava seu projeto. Nós infelizmente não sabemos o que aconteceu (senão teríamos contado!)

Se você pode confiavelmente fazer com que esse erro aconteça, nós apreciaríamos se você pudesse [abrir uma issue](https://astro.build/issues/).
