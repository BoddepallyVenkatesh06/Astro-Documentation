> O esquema de uma coleção de conteúdo não deve conter `slug` já que é reservado para a geração de slugs. Remova-o do esquema da coleção `NOME_COLEÇÃO`.

## O que deu errado?

O esquema de uma coleção de conteúdo não deve conter o campo `slug`. Ele é reservado pelo Astro para gerar slugs das entradas. Remova o campo `slug` do seu esquema ou escolha um nome diferente.
