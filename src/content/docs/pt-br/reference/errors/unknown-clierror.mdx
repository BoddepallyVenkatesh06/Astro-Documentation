## O que deu errado?

Astro encontrou um erro desconhecido enquanto iniciava um de seus comandos de interface de linha de comando. A mensagem de erro deve conter mais informações.

Se você pode confiavelmente fazer com que esse erro aconteça, nós apreciaríamos se você pudesse [abrir uma issue](https://astro.build/issues/).
