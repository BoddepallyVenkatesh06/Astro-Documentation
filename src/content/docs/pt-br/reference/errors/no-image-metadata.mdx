> Não foi possível processar os metadados da imagem para `CAMINHO_IMAGEM`.

## O que deu errado?
O Astro não conseguiu processar os metadados de uma imagem que você importou. Isso geralmente ocorre devido a uma imagem corrompida ou malformada, e reexportar a imagem do seu editor de imagens pode resolver esse problema.
