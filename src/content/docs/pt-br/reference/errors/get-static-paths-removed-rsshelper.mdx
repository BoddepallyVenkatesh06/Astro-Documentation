> **GetStaticPathsRemovedRSSHelper**: O utilitário de RSS foi removido de `getStaticPaths`. Tente o novo pacote @astrojs/rss no lugar.

## O que deu errado?

`getStaticPaths` não expõe mais um utilitário para gerar um feed RSS. Nós recomendamos migrar para a integração [@astrojs/rss](/pt-br/guides/rss/#configurando-astrojsrss) no lugar.
