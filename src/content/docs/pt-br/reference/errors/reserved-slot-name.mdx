> **ReservedSlotName**: Incapaz de criar um slot chamado `NOME_SLOT`. `NOME_SLOT` é um nome de slot reservado. Por favor atualize o nome desse slot.

## O que deu errado?

Certas palavras não podem ser utilizadas como nomes de slot por já serem utilizados internamente.
