> **Mensagens de erro de exemplo:**<br/>
InvalidComponentArgs: Argumentos inválidos passados ao componente `<MeuComponenteAstro>`.

## O que deu errado?

Componentes Astro não podem ser renderizados manualmente por uma chamada de função, como `Componente()` ou `{itens.map(Componente)}`. Opte pela sintaxe de componente `<Componente />` ou `{itens.map(item => <Componente {...item} />)}`.
