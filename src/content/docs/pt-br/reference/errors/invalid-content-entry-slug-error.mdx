> `NOME_COLEÇÃO` → `ID_ENTRADA` tem um slug inválido. `slug` deve ser uma string.

## O que deu errado?

Uma entrada em `src/content/` tem um `slug` inválido. Esse campo é reservado para a geração de slugs de entradas, e deve ser uma string quando presente.
