> **Mensagens de erro de exemplo:**<br/>
Não foi possível fazer parse do frontmatter em **blog** → **postagem.md**<br/>
"titulo" é obrigatório.<br/>
"data" deve ser uma data válida.

## O que deu errado?

O frontmatter de um documento Markdown em `src/content/` não corresponde ao esquema de sua coleção.
Certifique-se de que todos os campos obrigatórios estão presentes e possuem os tipos corretos.
Você pode verificar contra o esquema da coleção em seu arquivo `src/content/config.*`.
Veja a [documentação de coleções de conteúdo](/pt-br/guides/content-collections/) para mais informação.
