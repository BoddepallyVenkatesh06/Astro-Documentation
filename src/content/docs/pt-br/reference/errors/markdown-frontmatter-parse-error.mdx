> **Mensagens de erro de exemplo:**<br/>
não é possível ler um par de mapeamento implícito; está faltando um dois pontos<br/>
fim de stream inesperado em uma scalar entre aspas duplas<br/>
não é possível ler uma entrada de mapeamento de bloco; uma chave multilinha não pode ser ser uma chave implícita

## O que deu errado?

Astro encontrou um erro enquanto fazia parse do frontmatter do seu arquivo Markdown.
Isso é geralmente causado por um erro na sintaxe, como esquecer um dois pontos ou uma aspas final.
