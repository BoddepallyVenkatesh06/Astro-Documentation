> **Mensagens de erro de exemplo:**<br/>
InvalidPrerenderExport: Uma exportação de `prerender` foi detectada, mas seu valor não pode ser analisado estaticamente.

## O que deu errado?

A funcionalidade `prerender` apenas suporta um subconjunto de JavaScript válido — certifique-se de usar exatamente `export const prerender = true` para que nosso compilador possa detectar essa diretiva em tempo de build. Variáveis e declarações `let` e `var` não são suportadas.
