> Falta el atributo de ancho y alto para `IMAGE_URL`. Cuando se usan imágenes remotas, siempre se requieren ambas dimensiones para evitar el desplazamiento acumulativo de la capa (CLS).

## ¿Qué salió mal?
Para imágenes remotas, no se pueden inferir `width` y `height` del archivo original. Por lo tanto, para evitar el CLS, siempre se requieren esas dos propiedades.

Si tu imagen está dentro de tu carpeta `src`, probablemente quieras importarla en su lugar. Consulta [la guía de importaciones para obtener más información](/es/guides/imports/#otros-recursos).
