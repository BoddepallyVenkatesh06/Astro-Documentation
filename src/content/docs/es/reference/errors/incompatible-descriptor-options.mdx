> **IncompatibleDescriptorOptions**: Solo se puede especificar uno de `densities` o `widths`. En la mayoría de los casos, probablemente solo querrás usar `widths` si necesitas anchos específicos.

## ¿Qué salió mal?

Solo se puede especificar uno de `densities` o `widths`. Estos atributos se utilizan para construir un atributo `srcset`, el cual no puede contener tanto descriptores `x` como `w` al mismo tiempo.
