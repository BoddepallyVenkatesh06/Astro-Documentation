> **UnsupportedConfigTransformError**: Las funciones `transform()` en tu configuración de contenido deben devolver un JSON válido o tipos de datos compatibles con la biblioteca devalue (incluidas las fechas, los Maps y los Sets).<br/>Error completo: PARSE_ERROR (E09008)

## ¿Qué salió mal?
Las funciones `transform()` en tu configuración de contenido deben devolver un JSON válido o tipos de datos compatibles con la biblioteca devalue (incluidas las fechas, los Maps y los Sets).
