> **NoAdapterInstalled**: No se puede usar `output: 'server'` o `output: 'hybrid'` sin un adaptador. Instale y configure el adaptador adecuado para el despliegue final.

## ¿Qué salió mal?

Para usar el renderizado en el servidor, se debe instalar un adaptador para que Astro sepa cómo generar la salida adecuada para la plataforma de despliegue específica.
