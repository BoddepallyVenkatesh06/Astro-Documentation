> **ImageNotFound**: No se pudo encontrar la imagen solicitada `RUTA_DE_LA_IMAGEN`. ¿Existe?

## ¿Qué salió mal?

Astro no pudo encontrar una imagen que importaste. A menudo, esto se debe simplemente a un error tipográfico en la ruta.

Las imágenes en Markdown son relativas al archivo actual. Para hacer referencia a una imagen que se encuentra en la misma carpeta que el archivo `.md`, la ruta debe comenzar con `./`.
