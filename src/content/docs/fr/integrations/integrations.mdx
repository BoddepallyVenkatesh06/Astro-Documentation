## Dépots / Kits de démarrage

Vous pouvez démarrer un nouveau projet Astro basé sur un dépôt github existant en passant un argument `--template` à la commande `create astro`.

<PackageManagerTabs>
  <Fragment slot="npm">
  ```shell
  npm create astro@latest -- --template <nom-d-utilisateur-github>/<depot-github>
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```shell
  pnpm create astro@latest --template <nom-d-utilisateur-github>/<depot-github>
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```shell
  yarn create astro --template <nom-d-utilisateur-github>/<depot-github>
  ```
  </Fragment>
</PackageManagerTabs>
