> **ExpectedImage**: Expected `src` property for `getImage` or `<Image />` to be either an ESM imported image or a string with the path of a remote image. Received `SRC` (type: `TYPEOF_OPTIONS`).<br/><br/>Full serialized options received: `FULL_OPTIONS`.

## Qu'est-ce qui ne va pas ?
La propriété `src` d'une image n'est pas valide. Le composant Image exige que l'attribut `src` soit une image importée par ESM ou une chaîne de caractères. Ceci est également vrai pour le premier paramètre de `getImage()`.

```astro
---
import { Image } from "astro:assets";
import myImage from "../assets/my_image.png";
---

<Image src={myImage} alt="..." />
<Image src="https://example.com/logo.png" width={300} height={300} alt="..." />
```

Dans la plupart des cas, cette erreur se produit lorsque la valeur passée à `src` est indéfinie.
