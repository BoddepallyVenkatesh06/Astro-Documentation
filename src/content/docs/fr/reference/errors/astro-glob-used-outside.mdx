> **AstroGlobUsedOutside**: `Astro.glob(GLOB_STR)` can only be used in `.astro` files. `import.meta.glob(GLOB_STR)` can be used instead to achieve a similar result.
## Qu'est-ce qui ne va pas ?
`Astro.glob()` ne peut être utilisé que dans les fichiers `.astro`. Vous pouvez utiliser [`import.meta.glob()`](https://vitejs.dev/guide/features.html#glob-import) à la place pour obtenir le même résultat.
