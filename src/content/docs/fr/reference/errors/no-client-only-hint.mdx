> **NoClientOnlyHint**: Unable to render `COMPONENT_NAME`. When using the `client:only` hydration strategy, Astro needs a hint to use the correct renderer.

## Qu'est-ce qui ne va pas ?
Les composants `client:only` ne sont pas exécutés sur le serveur, Astro ne sait donc pas (et ne peut pas deviner) quel moteur de rendu utiliser et a besoin d'un indice. Par exemple :

```astro
<SomeReactComponent client:only="react" />
```
