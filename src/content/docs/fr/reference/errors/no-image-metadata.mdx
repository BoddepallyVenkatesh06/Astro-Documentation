> Could not process image metadata for `IMAGE_PATH`.

## Qu'est-ce qui ne va pas ?
Astro n'a pas pu traiter les métadonnées d'une image que vous avez importée. Cela est souvent dû à une image corrompue ou mal formée. La réexportation de l'image à partir de votre éditeur d'images peut résoudre ce problème.
