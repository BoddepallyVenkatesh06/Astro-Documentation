> **ImageNotFound**: Could not find requested image `IMAGE_PATH`. Does it exist?

## Qu'est-ce qui ne va pas ?
Astro n'a pas trouvé l'image que vous avez importée. Souvent, cela est simplement dû à une faute de frappe dans le chemin d'accès.

Les images en Markdown sont relatives au fichier courant. Pour faire référence à une image située dans le même dossier que le fichier `.md`, le chemin doit commencer par `./`
