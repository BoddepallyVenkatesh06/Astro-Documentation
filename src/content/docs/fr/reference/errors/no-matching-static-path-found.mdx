> **NoMatchingStaticPathFound**: A `getStaticPaths()` route pattern was matched, but no matching static path was found for requested path `PATH_NAME`.

## Qu'est-ce qui ne va pas ?
Une [route dynamique](/fr/core-concepts/routing/#routes-dynamiques) a été trouvée, mais aucun chemin correspondant n'a été trouvé pour les paramètres demandés. Cela est souvent dû à une faute de frappe dans le chemin généré ou dans le chemin demandé.

-  [getStaticPaths()](/fr/reference/api-reference/#getstaticpaths)
