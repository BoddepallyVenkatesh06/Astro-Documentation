> **IncompatibleDescriptorOptions**: Only one of `densities` or `widths` can be specified. In most cases, you'll probably want to use only `widths` if you require specific widths.

## Qu'est-ce qui ne va pas ?
Un seul des attributs `densities` ou `widths` peut être spécifié. Ces attributs sont utilisés pour construire un attribut `srcset`, qui ne peut pas avoir les deux descripteurs `x` et `w`.
