> **ExpectedImageOptions**: Expected getImage() parameter to be an object. Received `OPTIONS`.

## Qu'est-ce qui ne va pas ?
`getImage()`'s le premier paramètre doit être un objet contenant les différentes propriétés à appliquer à votre image.

```ts
import { getImage } from "astro:assets";
import myImage from "../assets/my_image.png";

const optimizedImage = await getImage({src: myImage, width: 300, height: 300});
```

Dans la plupart des cas, cette erreur se produit parce que les paramètres ont été transmis directement au lieu d'être placés à l'intérieur d'un objet.
