> **StaticClientAddressNotAvailable**: `Astro.clientAddress` is only available when using `output: 'server'` or `output: 'hybrid'`. Update your Astro config if you need SSR features.

## Qu'est-ce qui ne va pas ?
La propriété `Astro.clientAddress` n'est disponible que lorsque [Server-side rendering](/fr/guides/server-side-rendering/) est activé.

Pour obtenir l'adresse IP de l'utilisateur en mode statique, différentes API telles que [Ipify](https://www.ipify.org/) peuvent être utilisées dans un [Script côté client](/fr/guides/client-side-scripts/) ou il peut être possible d'obtenir l'adresse IP de l'utilisateur en utilisant une fonction sans serveur hébergée par votre fournisseur d'hébergement.
