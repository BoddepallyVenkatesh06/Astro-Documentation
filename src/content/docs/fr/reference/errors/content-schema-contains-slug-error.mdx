> **ContentSchemaContainsSlugError**: Un schéma de collection de contenu ne doit pas contenir `slug` puisqu'il est réservé à la génération de slug. Supprimez ceci de votre schéma de collection COLLECTION_NAME.

## Qu'est-ce qui a mal tourné ?
Un schéma de collection ne doit pas contenir le champ `slug`. Ce champ est réservé par Astro pour la génération d'entrées. Supprimez le champ `slug` de votre schéma, ou choisissez un nom différent.
