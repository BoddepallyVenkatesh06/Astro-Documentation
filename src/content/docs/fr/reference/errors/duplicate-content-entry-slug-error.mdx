> `COLLECTION_NAME` contains multiple entries with the same slug: `SLUG`. Slugs must be unique.

## Qu'est-ce qui ne va pas ?
Les entrées de la collection de contenus doivent avoir des slugs uniques. Les doublons sont souvent dus à la propriété `slug` frontmatter.
