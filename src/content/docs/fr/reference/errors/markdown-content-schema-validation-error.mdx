> **Exemple de message d'erreur :**<br/>
Could not parse frontmatter in **blog** → **post.md**<br/>
"title" is required.<br/>
"date" must be a valid date.

## Qu'est-ce qui ne va pas ?
Le frontmatter d'un document Markdown dans `src/content/` ne correspond pas au schéma de sa collection.
Assurez-vous que tous les champs obligatoires sont présents, et que tous les champs sont du bon type.
Vous pouvez vérifier le schéma de la collection dans votre fichier `src/content/config.*`.
Voir la [Documentation sur les collections de contenu](/fr/guides/content-collections/) pour plus d'informations.
