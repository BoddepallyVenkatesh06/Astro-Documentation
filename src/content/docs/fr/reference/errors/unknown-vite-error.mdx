## Qu'est-ce qui ne va pas ?
Vite a rencontré une erreur inconnue lors du rendu de votre projet. Nous ne savons malheureusement pas ce qui s'est passé (sinon nous vous le dirions !)

Si vous pouvez provoquer cette erreur de manière fiable, nous vous serions reconnaissants de bien vouloir [ouvrir un issue](https://astro.build/issues/).

**Voir aussi :**
-  [Guide de dépannage de Vite](https://vitejs.dev/guide/troubleshooting.html)
