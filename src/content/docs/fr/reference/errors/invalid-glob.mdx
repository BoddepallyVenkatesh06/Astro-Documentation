> **InvalidGlob**: Invalid glob pattern: `GLOB_PATTERN`. Glob patterns must start with './', '../' or '/'.

## Qu'est-ce qui ne va pas ?
Astro a rencontré un schéma (pattern) global non valide. Cela est souvent dû au fait que le schéma global n'est pas un chemin d'accès valide.
