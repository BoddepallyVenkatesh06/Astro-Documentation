> Missing width and height attributes for `IMAGE_URL`. When using remote images, both dimensions are always required in order to avoid cumulative layout shift (CLS).

## Qu'est-ce qui ne va pas ?
Pour les images distantes, les propriétés `width` et `height` ne peuvent pas être déduites du fichier original. C'est pourquoi, afin d'éviter le CLS (Cumulative Layout Shift), ces deux propriétés sont toujours requises.

Si votre image se trouve dans votre dossier `src`, vous avez probablement voulu l'importer à la place. Voir [le guide des importations pour plus d'informations](/fr/guides/imports/#autres-ressources).
