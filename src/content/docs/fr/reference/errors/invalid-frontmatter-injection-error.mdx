> **InvalidFrontmatterInjectionError** : Un plugin remark ou rehype a tenté d'injecter du frontmatter invalide. Assurez-vous que "astro.frontmatter" est configuré comme un objet JSON valide qui n'est pas `null` ou `undefined`.

## Qu'est-ce qui a mal tourné ?

Un plugin remark ou rehype a tenté d'injecter du frontmatter invalide. Cela se produit lorsque "astro.frontmatter" est `null`, `undefined` ou un objet JSON invalide.
