Ici, vous allez connecter votre dépôt GitHub à Netlify. Netlify utilisera ce projet pour construire et déployer votre site en direct sur le web chaque fois que vous validerez un changement dans votre code.

:::tip[Nous utiliserons...]
Ce tutoriel utilisera **Netlify**, mais vous pouvez utiliser le service d'hébergement de votre choix pour déployer votre site sur Internet.
:::

## Créez un nouveau site Netlify

1. Créez un compte gratuit sur [Netlify](https://netlify.com) si vous n'en avez pas déjà un.

    Prenez note de votre nom d'utilisateur. Vous pourrez afficher votre tableau de bord et tous les sites que vous créez à l'adresse `https://app.netlify.com/teams/nom-d-utilisateur`

2. Cliquez sur <kbd>Ajouter un nouveau site</kbd> > <kbd>Importer un projet existant</kbd>.

    On vous demandera de vous connecter à un fournisseur Git. Choisissez GitHub et suivez les étapes à l'écran pour authentifier votre compte GitHub. Ensuite, choisissez le dépôt GitHub de votre projet Astro dans la liste fournie.

3. À la dernière étape, Netlify vous montrera les paramètres de votre site. Les valeurs par défaut devraient être correctes pour votre projet Astro, vous pouvez donc faire défiler vers le bas et cliquer sur <kbd>Déployer le site</kbd>.

Félicitations, vous avez un site Astro !

## Changez le nom de votre projet

Sur la page d'aperçu de votre site dans Netlify, vous verrez le nom de projet généré aléatoirement, ainsi que l'URL de votre site sous la forme `https://nom-du-projet-123456.netlify.app`. Vous pouvez changer le nom de votre projet pour quelque chose de plus mémorable, et cela mettra automatiquement à jour votre URL.

## Visitez votre nouveau site

Cliquez sur l'URL dans les paramètres de votre site, ou saisissez-la dans une fenêtre de navigateur pour afficher votre nouveau site web.
