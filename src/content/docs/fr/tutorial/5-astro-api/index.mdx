Maintenant que vous avez quelques billets de blog, utilisons l'API d'Astro pour travailler avec vos fichiers !

## Où allons-nous ?

Dans cette unité, vous allez donner un coup de fouet à votre blog avec une page d'index, des pages de balises et un flux RSS.

En chemin, vous apprendrez à utiliser :
- `Astro.glob()` pour accéder aux données à partir des fichiers de votre projet
- `getStaticPaths()` pour créer plusieurs pages (itinéraires) en une seule fois
- Le paquet RSS Astro pour créer un flux RSS
