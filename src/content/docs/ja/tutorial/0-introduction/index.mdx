<Lede> このチュートリアルでは、完全に機能するブログをゼロから作成しながら、Astroの主要な機能を学んでいきます！🚀 </Lede>

その過程で、以下のことをおこないます。
- 開発環境の準備
- ウェブサイトのページとブログ記事の作成
- Astroコンポーネントの使用
- ローカルファイルの読み込みと操作
- サイトへのインタラクションの追加
- サイトのウェブ上へのデプロイ

これから作成するものを事前に見たい場合は、[GitHub](https://github.com/withastro/blog-tutorial-demo)や[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro)でプロジェクトの最終版を確認してください。

:::note
完成済みのAstroサイトを通じてAstroの学習を始めたければ、https://astro.new でスターターテンプレートを選択してオンラインエディターで編集してみましょう。
:::
