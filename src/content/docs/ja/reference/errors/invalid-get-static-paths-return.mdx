> **InvalidGetStaticPathsReturn**: `getStaticPaths`で無効な型が返されました。`array`である必要がありますが、`RETURN_TYPE`が返されました。

## 何が問題か？

`getStaticPaths`の戻り値はオブジェクトの配列である必要があります。

```ts title="pages/blog/[id].astro"
export async function getStaticPaths() {
	return [ // <-- 配列
		{ params: { slug: "ブログ" } },
		{ params: { slug: "概要" } }
	];
}
```
