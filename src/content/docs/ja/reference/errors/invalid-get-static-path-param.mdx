> **InvalidGetStaticPathParam**: `getStaticPaths`のパスに無効なparamsが指定されました。`object`である必要がありますが、`PARAM_TYPE`が指定されました。

## 何が問題か？

`getStaticPaths`の戻り値（オブジェクトの配列）の`params`プロパティもオブジェクトである必要があります。

```astro title="pages/blog/[id].astro"
---
export async function getStaticPaths() {
	return [
		{ params: { slug: "ブログ" } },
		{ params: { slug: "概要" } }
	];
}
---
```
