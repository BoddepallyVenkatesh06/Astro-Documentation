> **NoClientOnlyHint**: `COMPONENT_NAME`をレンダリングできません。`client:only`のハイドレーション戦略を使用する場合、Astroは適切なレンダラーを使用するためのヒントが必要です。

## 何が問題か？

`client:only`コンポーネントはサーバー上で実行されないため、Astroはどのレンダラーを使用するかを知る（あるいは推測する）ことができず、以下のようなヒントを必要とします。

```astro
	<SomeReactComponent client:only="react" />
```
