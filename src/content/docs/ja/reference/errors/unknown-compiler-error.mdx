> 不明なコンパイラのエラー。

## 何が問題か？

Astroは、ファイルをコンパイルする際に不明なエラーに遭遇しました。ほとんどの場合、これはあなたのせいではなく、コンパイラの問題です。

関連するIssueがまだない場合は、[Issueを作成](https://astro.build/issues/compiler)してください。
